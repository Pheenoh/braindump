## Commands
---
### DDL Commands
Data Definition Language commands:

- CREATE
- DROP
- ALTER
- TRUNCATE
- COMMENT
- RENAME

### DQL Commands
Data Query Language commands:

- SELECT

### DML Commands
Data Manipulation Language commands:

- INSERT
- UPDATE
- DELETE
- LOCK
- CALL
- EXPLAIN PLAN

### DCL Commands
Data Control Language commands:

- GRANT
- REVOKE


### TCL Commands
Transaction Control Language commands:

- COMMIT
- ROLLBACK
- SAVEPOINT
- SET TRANSACTION
